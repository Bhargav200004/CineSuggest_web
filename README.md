# 🎬 CineSuggest - Your Personal Movie Companion

Ever spent hours scrolling through streaming services trying to find something good to watch? We've all been there. That's why we built CineSuggest - your friendly neighborhood movie discovery app that helps you find your next favorite film without the endless scrolling.

## ✨ What Makes CineSuggest Special?

CineSuggest isn't just another movie database. It's a smart, intuitive platform that learns what you love and helps you discover hidden gems you might have missed. Here's what makes it awesome:

### 🎯 Personalized Discovery
- **Smart Recommendations**: Get movie suggestions based on your unique taste and viewing history
- **Advanced Filtering**: Filter by genre, year, rating, and more to find exactly what you're in the mood for
- **Powerful Search**: Search movies by title, genre, or even year - we've got you covered

### ❤️ Your Personal Movie Collection
- **Favorites System**: Build your personal collection of movies you love
- **Rating System**: Rate movies on a 5-star scale and keep track of what you thought
- **Watch History**: Never forget what you've watched and how you rated it

### 🎨 Beautiful & Intuitive Design
- **Modern UI**: Clean, responsive design that looks great on any device
- **Smooth Interactions**: Hover effects, animations, and transitions that make browsing enjoyable
- **Dark Theme**: Easy on the eyes during those late-night movie sessions

## 🚀 Getting Started

### Prerequisites
- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Bhargav200004/CineSuggest_web.git
   cd CineSuggest_web
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` and start exploring!

## 🎮 How to Use

### For New Users
1. **Sign up** for a free account to unlock all features
2. **Rate a few movies** to help our algorithm understand your taste
3. **Browse recommendations** tailored specifically for you
4. **Build your favorites list** by clicking the heart icon on movies you love

### For Movie Enthusiasts
- **Discovery Page**: Explore trending movies, search for specific films, or use our advanced filters
- **Movie Details**: Dive deep into movie information, cast, ratings, and more
- **Your Profile**: Track your ratings, favorites, and viewing history all in one place
- **Personalized Tabs**: Switch between Browse, Recommendations, and Random discoveries

## 🛠️ Tech Stack

We built CineSuggest with modern web technologies to ensure a smooth, responsive experience:

- **Frontend**: React 18 with modern hooks and patterns
- **Styling**: TailwindCSS for beautiful, responsive designs
- **Routing**: React Router for seamless navigation
- **State Management**: Context API and custom hooks
- **Build Tool**: Vite for lightning-fast development
- **Icons**: Lucide React for crisp, scalable icons
